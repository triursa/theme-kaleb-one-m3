Code Export

Copy theme integration snippets for your project.

CDN (HTML / Static pages)

Copy
<!-- Load theme --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/triursa/theme-kaleb-one@main/dist/theme-all.css"> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/triursa/theme-kaleb-one@main/dist/components.css"> <!-- Load fonts --> <link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet"> <!-- Activate palette --> <html data-theme="obsidian">

npm (Build-tool projects)

Copy
npm install @kaleb-one/theme // Import in JS: import '@kaleb-one/theme/css/all'; import '@kaleb-one/theme/css/components'; // Or single palette: import '@kaleb-one/theme/css/obsidian';

Tailwind Preset

Copy
// tailwind.config.js module.exports = { presets: [require('@kaleb-one/theme/tailwind')], };
